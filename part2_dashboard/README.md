# Part 2: Business Intelligence (Campaign Performance Dashboard)

This dashboard was built using **Streamlit** to provide business users with interactive insights on campaign performance, lead outcomes, and ad spend efficiency. Users can filter by business user, project, and date range to explore KPIs, trends, segmentations, and download filtered leads data.

## How to Run

1. Make sure you have Python 3.9+ installed.
2. Create and activate a virtual environment
   
    ```bash
   python -m venv venv
   venv\Scripts\activate
   if you faced issue. pls run this before activation : Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
    
     ```
3. Install the required packages:
   
   ```bash
   pip install -r requirements.txt

     ```

4. Run the Streamlit app:

    ```bash
       streamlit run app.py
      ```
